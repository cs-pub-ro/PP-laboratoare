\texttt{type} ne permite definirea unui sinonim de tip, similar cu
\texttt{typedef} din C.

\begin{lstlisting}[language=Haskell, numbers=none]
type Point = (Int, Int)

p :: Point
p = (2, 3)
\end{lstlisting}
\end{minipage}

\begin{tikzpicture}
\node [mybox_orange] (box){%
\begin{minipage}{0.3\textwidth}

\texttt{newtype} este similar cu data, cu diferența că ne \mbox{permite} crearea
unui tip de date cu un singur constructor, pe baza altor tipuri de date
existente.

\begin{lstlisting}[language=Haskell, numbers=none]
newtype Celsius = MakeCelsius Float deriving Show

newtype Fahrenheit = MakeFahrenheit Float
    deriving Show

celsiusToFahrenheit :: Celsius -> Fahrenheit
celsiusToFahrenheit (MakeCelsius c) =
    MakeFahrenheit $ c * 9/5 + 32
\end{lstlisting}
\end{minipage}
};

\node[fancytitle_orange, right=10pt] at (box.north west) {\texttt{newtype}};
\end{tikzpicture}

\begin{tikzpicture}
\node [mybox_cerise] (box){%
\begin{minipage}{0.3\textwidth}

\texttt{data} permite definirea de noi tipuri de date algebrice.

\begin{lstlisting}[language=Haskell, numbers=none]
data PointT = PointC Double Double deriving Show
\end{lstlisting}

\vspace{0.6em}
Tipuri enumerate.

\begin{lstlisting}[language=Haskell, numbers=none]
data Colour = Red | Green | Blue | Black
    deriving Show
nonColour :: Colour -> Bool
nonColour Black = True
nonColour _     = False
\end{lstlisting}

\vspace{0.6em}
Tipuri înregistrare.

\begin{lstlisting}[language=Haskell, numbers=none]
data PointT = PointC
    { px :: Double
    , py :: Double
    } deriving Show
px (PointC x _) = x
py (PointC _ y) = y
\end{lstlisting}

\vspace{0.6em}
Tipuri parametrizate.

\begin{lstlisting}[language=Haskell, numbers=none]
data Maybe a = Just a | Nothing
    deriving (Show, Eq, Ord)
maybeHead :: [a] -> Maybe a
maybeHead (x : _) = Just x
maybeHead _       = Nothing
\end{lstlisting}

\vspace{0.6em}
Tipuri recursive.

\begin{lstlisting}[language=Haskell, numbers=none]
data List a = Void | Cons a (List a) deriving Show

data Natural = Zero | Succ Natural deriving Show
\end{lstlisting}
\end{minipage}
};

\node[fancytitle_cerise, right=10pt] at (box.north west) {\texttt{data}};
\end{tikzpicture}
